# Blueprint: MBTI 테스트 웹 애플리케이션

## 프로젝트 개요

이 프로젝트는 사용자가 간단한 질문을 통해 자신의 MBTI 성격 유형을 알아볼 수 있는 웹 애플리케이션입니다. 테스트 결과와 함께 각 MBTI 유형에 대한 설명을 제공하며, 16가지 성격 유형 각각에 대한 상세 페이지를 포함하고 있습니다. 또한, 다크 모드 기능을 지원하여 사용자 경험을 향상시키고, 제휴 문의 양식을 통해 비즈니스 기회를 창출할 수 있도록 설계되었습니다. Google 애널리틱스와 Microsoft Clarity를 추가하여 사용자 행동을 심층적으로 분석하고 서비스를 개선하는 데 활용합니다.

## 구현된 기능

*   **MBTI 테스트 기능:** `test.html` 페이지에서 제공되는 질문을 통해 사용자의 MBTI 유형을 판별합니다.
*   **결과 페이지:** `result.html`에서 테스트 완료 후 사용자의 MBTI 유형과 간단한 설명, 상세 페이지 링크를 보여줍니다.
*   **16가지 성격 유형 상세 페이지:** 각 MBTI 유형(ISTJ, ISFJ, INFJ, INTJ, ISTP, ISFP, INFP, INTP, ESTP, ESFP, ENFP, ENTP, ESTJ, ESFJ, ENFJ, ENTJ)에 대한 상세 정보를 제공하는 별도의 HTML 페이지를 제공합니다.
*   **새로운 디자인 및 다크 모드:** 통통 튀는 디자인과 다크 모드 기능을 모든 페이지에 적용하여 일관성 있는 사용자 경험을 제공합니다.
*   **파비콘:** 웹사이트의 모든 페이지에 파비콘을 적용하여 브라우저 탭에서 쉽게 식별할 수 있도록 했습니다.
*   **개인정보처리방침 및 About 페이지:** 웹사이트의 신뢰도를 높이기 위한 개인정보처리방침 및 About 페이지를 포함합니다.
*   **반응형 웹 디자인:** 모바일 및 데스크톱 환경에서 최적화된 화면을 제공합니다.
*   **사용자 행동 분석 도구 연동:**
    *   **Google 애널리틱스:** 모든 페이지에 추적 코드를 삽입하여 방문자 데이터를 수집하고 분석합니다.
    *   **Microsoft Clarity:** 모든 페이지에 추적 코드를 삽입하여 히트맵, 세션 리코딩 등 사용자 상호작용을 시각적으로 분석합니다.
*   **SEO 및 검색 엔진 최적화:**
    *   **메타 태그:** 모든 페이지에 고유한 `title`, `description`, `keywords`를 추가하여 검색 결과 노출 가능성을 높였습니다.
    *   **Open Graph 및 Twitter 카드:** 소셜 미디어 공유 시 풍부한 미리보기를 제공하기 위해 Open Graph 및 Twitter 카드 메타 태그를 모든 페이지에 적용했습니다.
    *   **구조화된 데이터 (Schema.org):** 검색 엔진이 콘텐츠를 더 잘 이해하도록 `Article` 스키마를 사용하여 JSON-LD 형식의 구조화된 데이터를 추가했습니다.
    *   **캐노니컬 URL:** 중복 콘텐츠 문제를 방지하기 위해 모든 페이지에 `canonical` URL을 지정했습니다.

## 현재 변경 사항 계획

**요청:** 모든 페이지에 검색 엔진 최적화(SEO)를 적용해 주세요.

**계획:**

1.  **SEO 분석 및 키워드 리서치:** 각 페이지의 핵심 콘텐츠를 파악하고 관련 키워드를 선정합니다.
2.  **메타 태그 작성:** 각 페이지별로 최적화된 `title`, `description`, `keywords` 메타 태그를 작성합니다.
3.  **소셜 미디어 태그 추가:** Open Graph 및 Twitter 카드 태그를 추가하여 소셜 공유 시 가시성을 높입니다.
4.  **구조화된 데이터 추가:** Schema.org의 `Article` 타입을 사용하여 JSON-LD 형식의 구조화된 데이터를 추가합니다.
5.  **캐노니컬 URL 설정:** 모든 페이지에 캐노니컬 URL을 지정하여 중복 콘텐츠 문제를 해결합니다.
6.  **시맨틱 HTML 적용:** 콘텐츠 구조를 명확히 하기 위해 `<article>`과 같은 시맨틱 태그를 사용합니다.
7.  **모든 페이지에 적용:** 위의 변경 사항을 프로젝트의 모든 HTML 파일에 적용합니다.

**실행:**

*   `index.html`, `about.html`, `test.html`, `result.html`, `privacy-policy.html` 및 16개의 모든 MBTI 유형 페이지에 맞춤형 `title`, `description`, `keywords` 메타 태그를 추가했습니다.
*   모든 페이지에 Open Graph 및 Twitter 카드 태그를 구현하여 소셜 미디어 공유 효율을 높였습니다.
*   모든 페이지에 `Article` 타입의 JSON-LD 구조화된 데이터를 추가하여 검색 엔진의 콘텐츠 이해도를 높였습니다.
*   모든 페이지에 캐노니컬 URL을 설정했습니다.
*   콘텐츠 영역을 `<article>` 태그로 감싸 시맨틱 구조를 개선했습니다.

**결과:**

*   웹사이트의 모든 페이지가 검색 엔진에 더 친화적으로 변경되어 검색 순위 상승 및 오가닉 트래픽 증가가 기대됩니다.
*   소셜 미디어에 링크 공유 시 더욱 풍부하고 매력적인 미리보기가 제공됩니다.
